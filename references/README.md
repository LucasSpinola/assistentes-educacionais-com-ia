# Referências

Bibliografia do projeto, em formato BibTeX, no arquivo
[referencias.bib](referencias.bib).

## Regra de ouro

Toda referência precisa ser real e verificada. Nunca inclua uma citação inventada,
nem invente autores, anos ou títulos. Antes de adicionar uma entrada, confirme os
dados no Google Scholar, no arXiv ou pelo DOI.

## Como citar nas aulas

Na seção de Referências Científicas de cada aula, cite as entradas pela chave do
BibTeX. Por exemplo, ao falar de Transformers, use a entrada
`vaswani2017attention`. Sempre que possível, traga o link do arXiv ou o DOI para o
leitor encontrar a fonte.

## Como adicionar uma referência nova

1. Encontre o trabalho e confirme os metadados na fonte original.
2. Copie a citação no formato BibTeX, de preferência a versão oficial.
3. Padronize a chave no estilo `sobrenomeAnoPalavra`, por exemplo
   `lewis2020rag`.
4. Adicione o arXiv ou o DOI no campo apropriado.
5. Confira se a entrada não está duplicada.

## Conteúdo atual

A bibliografia reúne os trabalhos canônicos que sustentam a trilha, de Word
Embeddings e redes recorrentes a Transformers, LLMs, RAG e agentes, além das
referências específicas citadas em cada módulo.
